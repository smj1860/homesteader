# **App Name**: SustainPro AI

## Core Features:

- AI Project Guide: Generate step-by-step DIY instructions for various categories (landscaping, freezing/canning, auto, building, cooking, home remedies, general maintenance, self-sufficiency). The AI tool provides safety/PPE, timeframe estimates, tools/items, and 'pro tips.' Offers both 'by the book' and 'MacGyver' instructions, with the latter available only in paid tiers and including liability disclaimers.
- Contractor Recommendation Engine: An AI-powered tool to provide a top 3 list of recommended contractors (plumbing, electrical, home projects) based on user's zip code, aggregated Google and Yelp reviews, and parsing of reviewer keywords. A passive list is always available for home-related projects.
- Inventory Manager: Allow users to input their on-hand items via text or file/picture upload, managing their personal inventory of supplies and tools.
- User Authentication & Tiered Access: Enable free user signup. The free tier provides access to 'by the book' instructions with 2 project credits, triggering a pricing modal thereafter. Paid tiers (subscription $8/month, $80/year, or $3/project) offer unlimited access and 'MacGyver' instructions.
- Monetization Integrations: Integrate advertising, affiliate links for recommended tools/parts within project guides, and a system for selling contractor leads.
- AI Clarification & Feedback Loop: Users can submit questions or feedback on specific project steps, and an AI tool will provide personalized clarifications or alternative explanations.

## Style Guidelines:

- Primary color: An earthy, muted green (#5F7F3F) to convey nature, growth, and reliability.
- Background color: A soft, almost off-white with a subtle green tint (#F4F6ED) for a clean, natural, and airy feel.
- Accent color: A warm, bright yellow-green (#F0DE3F) reminiscent of sunlight or new growth, used for highlights and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-assisted, and functional look. Body font: 'Inter' (sans-serif) for clear readability of instructions and longer text passages.
- Utilize clear, instructional icons that have a slightly organic or hand-drawn touch, consistent with homesteading and DIY themes.
- Implement a clean, card-based layout for project instructions and content, ensuring good whitespace for clarity. Prioritize mobile-first responsiveness for on-the-go access during projects.
- Subtle and functional animations for transitions between project steps, button clicks, and modal displays, providing smooth user feedback without distraction.
